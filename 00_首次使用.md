# 首次使用 {{tenant 代号}}_AIPM-Orbit

> 这是一个**空模板 vault**。第一次拿到不知道从哪开始？跟着这页走。

## 0. 检查前置工具

- [ ] **Obsidian** 已安装，并把本目录用「Open folder as vault」打开
- [ ] **任一 AI Agent CLI** 已安装（任选其一即可）：
  - Claude Code（`claude` 命令）— 推荐，原生支持 slash command 和自动 memory
  - Codex CLI（`codex` 命令）— 读 `AGENTS.md`
  - Cursor / Gemini CLI / 通义灵码 等其他 agent — 读 `AGENTS.md` 或 `GEMINI.md`
- [ ] （强烈推荐）安装 Obsidian 插件：**Dataview** + **Folder Notes**

## 1. 让 AI 跑初始化向导

在终端 `cd` 到本 vault 目录，启动你的 AI Agent，按下面对应方式触发：

**Claude Code 用户**：

```
/初始化向导
```

**Codex / Cursor / Gemini / 其他 agent 用户**：

> 请按 `.claude/commands/初始化向导.md` 里的流程引导我完成本 vault 的首次初始化。

> ⚠️ memory 步骤（Step 1）是 Claude Code 特有的自动记忆机制；其他 agent 没有这套机制，跳过即可——通用方法论会在你日常对话中由 AI 读取 `99_系统/memory_seed/` 作为参考。

AI 会按 5 步引导你：

1. **装 memory（Claude Code 专属，其他 agent 跳过）** — 把通用方法论种子复制到 Claude Code 的物理 memory 路径
2. **介绍业务本体** — 填齐 `99_系统/公司/本体档案/` 五张表
3. **灌入历史资料** — 把现有 PRD / 架构图 / 业务规则丢进 `00_基线资料/`，AI 边读边沉淀
4. **建第一个项目（XM）** — 在 `10_项目/` 下建第一个 `XM01_<名称>/`，承接你当下手上的事
5. **沉淀第一个产品（CP，可延后）** — 在 `20_产品/` 下建第一个 `CP01_<名称>/`，接住 XM 沉淀

整个流程 30~60 分钟。中途可以喊「暂停 / 保存进度 / 跳过这步」。

## 3. 不想用向导？手动初始化

不喜欢被引导也可以自己来：

1. 把 `99_系统/memory_seed/` 内所有 `.md` 复制到 `~/.claude/projects/-<vault路径encode>/memory/`（路径转 slug 规则：绝对路径里的 `/` 替换成 `-`，前面再加一个 `-`）
2. 编辑 `99_系统/公司/本体档案/01-05*.md` 填本体上下文
3. 编辑 `00_AIPM-Orbit整体说明.md` 填业务背景
4. 按需在 `20_产品/` `10_项目/` 下手动建产品和项目目录

完整框架规则见 [README.md](./README.md) 和 [CLAUDE.md](./CLAUDE.md)。
