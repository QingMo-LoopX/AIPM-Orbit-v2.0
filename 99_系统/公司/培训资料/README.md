# 培训资料

公司给员工的能力建设资源。按岗位分类，岗位通用培训内容。

## 结构

```
培训资料/
├── 产品经理/
└── <未来其他岗位>/
```

## 学习机制

LLM 没有持久记忆，看完不会自动记住。所以学习流程是：

1. **一次性学习**：员工被触发学习某份资料时，读 PDF / 文档
2. **提炼工艺要点**：把可复用的要点归纳成结构化条目
3. **沉淀到工艺手册**：把要点写到 `员工档案/<员工名>/工艺手册/` 对应条目
4. **日常工作只读工艺手册**：不每次读原始培训资料

PDF 原件留作档案，工艺手册是消化产物。

## 触发学习

用户口令 「**学一下培训资料**」 或 「**学这份 PDF：<路径>**」 触发。

员工的反馈：
- 读完后报告归纳要点
- 列出沉淀到了哪些工艺手册条目
- 标记「**待人工确认**」给用户审核
