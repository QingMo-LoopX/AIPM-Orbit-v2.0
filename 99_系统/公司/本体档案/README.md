## 概念
- **本体**：当前 vault 服务的业务实体——去掉法人 / 股权 / 办公场所等工业时代外壳后，剩下的「业务目标 + 运转规则 + 知识沉淀 + 当前状态」
- **入本体**：AI 员工被唤醒时加载本目录的过程
- **跨本体**：员工换 vault 服务不同业务（工艺手册可继承，本体档案不可迁移）
